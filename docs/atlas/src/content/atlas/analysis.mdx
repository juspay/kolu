The **Analysis** index gathers notes that investigate how the system behaves:
a measurement, a comparison against an alternative, a root-cause dig that isn't
yet a fix. The output is a conclusion you can act on, with the evidence attached.

This note is an *index* (`moc: true`): a note is filed here by listing `analysis`
in its `parents`. The graph renders it large; the notes wired to it are its
cluster, and its card on the [Atlas](./index.html) lists them all.
